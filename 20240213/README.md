## 주제: AI시대에 DevOps가 나아갈 방향은?

### 주요 아젠다
- AI시대 관련 변화된 시대에 맞게 대응 할 방향
- DevOps 업무에 도움이 될만한 AI도구 및 방법론
- 이미 적용중인 AI를 활용한 엔지니어링 업무소개 등
